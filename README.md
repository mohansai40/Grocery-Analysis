# 🛒 BlinkIT Grocery Sales Analysis Dashboard

## 🚀 Project Overview

This project focuses on analyzing **BlinkIT grocery sales data** to extract meaningful insights and support business decision-making. Using **Excel and Power BI**, an interactive dashboard was created to track key performance indicators (KPIs) such as sales, ratings, and product visibility.

---

## 🎯 Objectives

* Analyze overall sales performance across different categories
* Identify top-performing products and outlets
* Track customer ratings and item visibility
* Build an interactive dashboard for business insights

---

## 📊 Key Features

* Developed **interactive dashboards** using Power BI and Excel
* Used **Pivot Tables and Pivot Charts** for dynamic data analysis
* Created KPIs such as:

  * Total Sales
  * Average Rating
  * Item Visibility
  * Number of Items Sold
* Applied **filters and slicers** for better user interaction
* Performed **data cleaning and preprocessing** to ensure accuracy

---

## 🛠️ Tools & Technologies

* Microsoft Excel (Pivot Tables, Charts, Data Cleaning)
* Power BI (Dashboard, Data Visualization, DAX)

---

## 📈 Insights Generated

* Identified **top-performing product categories and outlets**
* Analyzed impact of **item visibility on sales performance**
* Observed trends in **customer ratings and product demand**
* Provided actionable insights to improve sales strategy

---

## 📁 Dataset

* The dataset contains information about:

  * Item Type
  * Sales
  * Ratings
  * Item Visibility
  * Outlet Type and Location
